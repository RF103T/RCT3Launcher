# RCT3启动器 (WinForm版)

**此版本已经停止维护，[新版本](https://github.com/RF103T/RCT3LauncherV2)正在开发中。**

支持的游戏版本：
+ RollerCoaster Tycoon® 3 Platinum
+ RollerCoaster Tycoon® 3 Complete Edition **（部分支持）**

功能：
+ 启动游戏
+ 添加扩充 **（不完善）**
+ 添加音乐
+ 存档恢复
+ 游戏修复 **（适用于Platinum）**
+ 集成SweetFX设置 **（SweetFX太过老旧，已经停止支持）**
